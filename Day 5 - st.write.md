# st.write

## What is st.write

`st.write` - for writing text and arguments, as well as:

- prints strings, works like `st.markdown()`
- display a python `dict`
- display `pandas` dataframes
- also show plots/figures from `matplotlib`, `plotly`, `altair`
- and more
