# Day 3 - st.button

## What we're building

Right now, it is a super simple that conditionally prints things

1. By default, the app prints `Goodbye`
2. When the button is clicked, the apps shows a new message

This will help learn how to use conditional triggers in streamlit and display different content based on user behavior

## What we covered

1. Importing `streamlit as st` at the start of the code
2. Creating a header with `st.header`
3. Assigning a `st.button` with text inside it
4. if/else with streamlit button and outputting text with `st.write`
